# AI From Scratch

This repository documents my journey of learning Artificial Intelligence from first principles.

The goal is not only to use AI frameworks but to understand how modern AI models work internally by implementing them from scratch.

## Roadmap

- Linear Algebra
- Probability & Statistics
- Neural Networks
- Backpropagation
- Transformers
- Tiny GPT

## Tech Stack

- Python
- NumPy
- Matplotlib
- PyTorch (later)
